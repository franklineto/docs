Códigos de Status HTTP

by Franklin
Referência completa — RFC 9110 / IANA
1xx — Informacional
Código Nome Descrição
100 Continue Servidor recebeu os headers; cliente pode continuar enviando o body.
101 Switching Protocols Servidor aceita troca de protocolo (ex: HTTP → WebSocket).
102 Processing Servidor está processando; evita timeout em operações longas.
103 Early Hints Envia headers de pré-carregamento antes da resposta final.
2xx — Sucesso
Código Nome Descrição
200 OK comum Requisição bem-sucedida. Usado em GET, PUT, PATCH.
201 Created comum Recurso criado com sucesso. Retornado em POST.
202 Accepted Requisição aceita para processamento assíncrono.
203 Non-Authoritative Info Resposta de proxy com conteúdo modificado.
204 No Content comum Sucesso sem body. Usado em DELETE e PATCH sem retorno.
205 Reset Content Sucesso; cliente deve limpar o formulário.
206 Partial Content Entrega parcial de arquivo (download resumível, streaming).
207 Multi-Status Múltiplos status em um único response (WebDAV).
208 Already Reported Membro já listado em resposta DAV anterior.
226 IM Used Resposta é resultado de manipulação de instância.
3xx — Redirecionamento
Código Nome Descrição
300 Multiple Choices Múltiplas representações disponíveis para o recurso.
301 Moved Permanently comum URL mudou permanentemente. Browsers atualizam o bookmark.
302 Found comum Redirecionamento temporário. URL original permanece válida.
303 See Other Redireciona para outro recurso via GET (pós-POST).
304 Not Modified comum Recurso em cache ainda é válido. Sem body na resposta.
307 Temporary Redirect Redirecionamento temporário preservando o método HTTP.
308 Permanent Redirect Redirecionamento permanente preservando o método HTTP.
4xx — Erro do Cliente
Código Nome Descrição
400 Bad Request comum Requisição malformada, JSON inválido ou dados incorretos.
401 Unauthorized comum Autenticação necessária ou credenciais inválidas.
402 Payment Required Reservado para uso futuro. Algumas APIs usam para paywall.
403 Forbidden comum Autenticado mas sem permissão para o recurso.
404 Not Found comum Recurso não encontrado no servidor.
405 Method Not Allowed REST Método HTTP não suportado para esta rota.
406 Not Acceptable Servidor não suporta o formato solicitado no Accept.
407 Proxy Auth Required Autenticação necessária no proxy intermediário.
408 Request Timeout Cliente demorou demais para enviar a requisição completa.
409 Conflict REST Conflito de estado: duplicata, versão desatualizada.
410 Gone Recurso removido permanentemente (diferente do 404).
411 Length Required Header Content-Length obrigatório mas ausente.
412 Precondition Failed Condição nos headers (If-Match, If-None-Match) falhou.
413 Content Too Large Body da requisição excede o limite configurado.
Código Nome Descrição
414 URI Too Long URL excede o tamanho máximo aceito pelo servidor.
415 Unsupported Media Type Content-Type não suportado (ex: esperava JSON, recebeu XML).
416 Range Not Satisfiable Intervalo solicitado no header Range é inválido.
417 Expectation Failed Header Expect não pôde ser atendido pelo servidor.
418 I'm a Teapot Easter egg do RFC 2324. Servidor recusa fazer café.
421 Misdirected Request Requisição enviada para servidor que não pode respondê-la.
422 Unprocessable Entity REST Body bem-formado mas falha na validação de negócio.
423 Locked Recurso bloqueado por outro processo (WebDAV).
424 Failed Dependency Ação falhou por dependência de outra ação que falhou.
425 Too Early Servidor recusa processar early data (risco de replay).
426 Upgrade Required Cliente deve trocar de protocolo (ex: HTTP/1.1 → HTTP/2).
428 Precondition Required Requer header condicional (If-Match) para evitar conflito.
429 Too Many Requests comum Rate limit excedido. Retry-After indica quando tentar novamente.
431 Headers Too Large Headers excedem o tamanho máximo aceito pelo servidor.
451 Unavailable Legal Recurso bloqueado por ordem legal (referência a Fahrenheit 451).
5xx — Erro do Servidor
Código Nome Descrição
500 Internal Server Error comum Erro genérico no servidor. Indica bug ou exceção não tratada.
501 Not Implemented Servidor não suporta o método HTTP utilizado.
502 Bad Gateway comum Gateway recebeu resposta inválida do servidor de origem.
503 Service Unavailable comum Servidor indisponível: sobrecarga ou manutenção.
Código Nome Descrição
504 Gateway Timeout comum Gateway não recebeu resposta do upstream a tempo.
505 HTTP Version Not Supported Versão do protocolo HTTP não suportada pelo servidor.
507 Insufficient Storage Servidor sem espaço para completar a requisição (WebDAV).
508 Loop Detected Loop infinito detectado ao processar a requisição.
511 Network Auth Required Cliente precisa autenticar na rede (ex: portal Wi-Fi captive).
comum = mais frequentes em APIs REST  |  REST = uso semântico específico em APIs REST  |  Referência: RFC 9110 & IANA HTTP Status Code Registry
